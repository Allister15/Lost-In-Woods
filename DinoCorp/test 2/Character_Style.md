Use the following requirements prompt when generating assets for Lost in the Woods. It is written to produce a consistent visual style across characters and environments.
Character Asset Prompt
Lost in the Woods — Main Character
Create a stylized 2D geometric character for a game called "Lost in the Woods".

Art Style:
- Flat 2D vector art
- Geometric minimalist design
- Inspired by Monument Valley, Alto's Adventure, and modern storybook illustrations
- Clean shapes with minimal details
- No outlines
- Soft gradients allowed
- Consistent proportions suitable for reusable game assets

Character:
- Young adult airplane crashed lost in a mysterious forest
- Neutral expression with slight concern or curiosity
- Readable silhouette
- Front-facing standing pose
- Symmetrical design

Body Construction:
- Head made from simple circle or rounded polygon
- Torso made from trapezoids and rectangles
- Arms and legs made from simple geometric segments
- Minimal facial features
- Large expressive eyes
- No realistic details

Clothing:
- Hooded cloak or simple traveler outfit
- Earth-tone colors
- Forest survival aesthetic
- Small satchel or backpack
- Boots

Color Palette:
- Muted greens
- Dark browns
- Soft beige
- Desaturated blue accents

Lighting:
- Moonlit forest lighting
- Cool blue highlights
- Subtle ambient glow

Background:
- Transparent background
- Character asset only

Output:
- Game-ready character sprite
- Centered composition
- High resolution
- Consistent with a geometric fairy-tale forest world
  Environment Asset Prompt
  Lost in the Woods — Night Forest
  Create a stylized 2D geometric night forest environment for a game called "Lost in the Woods".

Art Style:
- Flat geometric vector illustration
- Minimalist shapes
- Atmospheric and mysterious
- Storybook fantasy aesthetic
- Consistent with geometric character assets

Environment Theme:
- Dense forest at night
- Mysterious but beautiful
- Slightly spooky rather than horror
- Fairy-tale atmosphere

Scene Elements:
- Geometric pine trees
- Triangular tree silhouettes
- Circular moon
- Angular rocks
- Layered forest depth
- Mist between trees
- Soft glowing fireflies
- Winding path disappearing into darkness

Lighting:
- Moonlight dominant
- Deep blue color palette
- Soft purple shadows
- Gentle fog glow
- Strong foreground-background separation

Color Palette:
- Midnight blue
- Forest green
- Dark teal
- Soft violet
- Pale moonlight white

Composition:
- Side-scrolling game background
- Multiple depth layers
- Open path through center
- Areas for future encounters

Mood:
- Curious
- Mysterious
- Magical
- Slightly dangerous
- Adventure-focused

Avoid:
- Realistic rendering
- Photorealism
- Gore
- Horror monsters
- Heavy texture noise

Output:
- Wide landscape background
- Suitable for narrative exploration game
- High resolution
- Consistent geometric visual language
  Style Guide Prompt (Use for Every Future Asset)
  All Lost in the Woods assets must follow the same visual language:

- Flat 2D geometric shapes
- Minimalist vector art
- Dark fairy-tale atmosphere
- Moonlit lighting
- Muted natural colors
- Soft gradients
- No realistic textures
- Clean silhouettes
- Readable at small sizes
- Consistent proportions
- Storybook adventure aesthetic
- Slightly spooky but not horror
  This style guide will help ensure your characters, NPCs (hunter, ghost, werewolf, peddler, grandma, dwarf), locations (swamp, pond, cliff, clearing), and UI all feel like they belong to the same world.
